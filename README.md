# IPL Match Data Analysis (2008–2024) 🏏

A comprehensive Exploratory Data Analysis (EDA) project on the **Indian Premier League (IPL)** dataset covering seasons **2008–2024**. This project uncovers trends in team performance, player achievements, venue statistics, toss decisions, match outcomes, and other key insights using Python.

---

## 📌 Project Overview

The Indian Premier League (IPL) is one of the world's most competitive T20 cricket leagues. This project analyzes IPL match data from **2008 to 2024** to identify patterns and generate meaningful insights through data visualization and statistical analysis.

The notebook includes:

* Data Cleaning & Preprocessing
* Exploratory Data Analysis (EDA)
* Team Performance Analysis
* Toss Impact Analysis
* Venue-wise Performance
* Player Performance Insights
* Winning Margin Analysis
* Interactive and Visual Charts

---

## 📂 Dataset

This project uses the **IPL Complete Dataset (2008–2020)** from Kaggle as the base dataset and has been extended with additional match records up to the **2024 IPL season** for analysis.

**Dataset Source:**
- Kaggle: https://www.kaggle.com/datasets/patrickb1912/ipl-complete-dataset-20082020

## 📌 Sample Insights

* Mumbai Indians and Chennai Super Kings have been among the most successful IPL franchises.
* Winning the toss does not always guarantee winning the match.
* Certain venues consistently favor chasing teams.
* A small group of players dominates the "Player of the Match" awards across seasons.
* Match-winning margins vary significantly depending on venue and innings.
